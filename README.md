MenuHotkeys is a "simple" Geometry Dash mod that uses the geode modding framework. It aims to add hotkeys to *most* of the games menu, so that you can use your keyboard to navigate the whole game, no need for a mouse!

Features:

-Hotkeys for almost every menu button in game, the game should be fully playable with just the keyboard now!

-Every key can be rebound via settings>options>keys.

-When you open the level search menu, it lets you start typing without having to click the search box.
